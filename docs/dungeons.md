# Dungeon System

Dungeons are high-risk, high-reward challenges.

## Purpose

- Gate difficult content
- Provide large XP and item rewards
- Act as progression milestones

## Unlock Conditions

- Minimum level required
- Optional rank requirement

## Difficulty Levels

- Easy
- Normal
- Hard
- Nightmare

Higher difficulty increases:
- XP multipliers
- Item drop chances
- Failure penalties

Dungeons are rule-based encounters and do not track progress themselves.
